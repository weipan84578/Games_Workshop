Controls

Left flipper: A / Left Arrow. Right flipper: D / Right Arrow. Launch: hold Space / Down Arrow, then release. Pause: Esc.

Scoring

Bumpers score 100, targets score 250, and ramps score 500. Fast repeated hits build combo and bonus multiplier.

Rules

You start with 3 balls. A weak launch returns to the plunger for another try. Only balls that have entered the table can drain.
