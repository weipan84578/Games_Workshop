# 光槍特訓 Shooting Game

純前端射擊訓練遊戲，依照 `shooting_game_spec.md` 實作。遊戲使用 HTML5、CSS3 與 Vanilla JavaScript，不需要 npm、打包工具或本機伺服器。

## 啟動方式

直接雙擊 `index.html`，或在瀏覽器中開啟該檔案即可開始遊戲。

## 玩法

- 點擊或觸控畫面射擊目標。
- 命中敵人得分，命中頭部可取得更高分。
- 誤傷人質會扣分並中斷連擊。
- 按 `R`、滑鼠右鍵、雙指觸控或右下角換彈按鈕可換彈。
- 完成關卡目標、生命歸零或倒數結束後進入結算畫面。

## 內容

- 主畫面、設定、排行榜、說明、關卡選擇、遊戲畫面與結算畫面
- 5 個一般關卡與無盡模式
- 敵人、人質、Boss、連擊、彈藥、換彈、倒數與計分系統
- Web Audio 程序音效
- `localStorage` 本機排行榜與設定保存

## 主要檔案

- `index.html`：遊戲入口
- `style/`：全域樣式、UI 與動畫
- `js/`：遊戲邏輯、狀態、關卡、音效、計分與儲存
- `shooting_game_spec.md`：完整規格書
