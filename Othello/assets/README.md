# Othello Frontend Assets

## English

### CSS

- `css/base.css`: design tokens, resets, global layout primitives, buttons, and form basics.
- `css/screens.css`: home screen, difficulty screen, and reusable screen/panel layouts.
- `css/game.css`: game board, cells, pieces, score panels, status bar, and in-game controls.
- `css/overlays-settings.css`: pause/result/rules overlays, settings UI, records UI, and toast.
- `css/responsive-motion.css`: keyframe animations and tablet/mobile media queries.

### JavaScript

- `js/constants.js`: board constants, directions, and AI position weights.
- `js/storage.js`: localStorage settings, saved game state, and records.
- `js/audio.js`: Web Audio API background music and sound effects.
- `js/game.js`: Othello rules, move execution, turn flow, scoring, and persistence.
- `js/ai.js`: Minimax AI, alpha-beta pruning, and board evaluation.
- `js/ui.js`: DOM rendering, screen navigation, settings binding, overlays, and responsive board sizing.
- `js/main.js`: application bootstrap.

## 日本語

### CSS

- `css/base.css`: デザイントークン、リセット、共通レイアウト、ボタン、フォームの基本スタイル。
- `css/screens.css`: ホーム画面、難易度選択画面、共通の画面およびパネルレイアウト。
- `css/game.css`: ゲーム盤、マス、石、スコアパネル、ステータスバー、対局中の操作部品。
- `css/overlays-settings.css`: 一時停止、結果、ルール説明のオーバーレイ、設定画面、戦績画面、トースト。
- `css/responsive-motion.css`: キーフレームアニメーション、タブレットおよびモバイル向けメディアクエリ。

### JavaScript

- `js/constants.js`: 盤面定数、探索方向、AI 用の位置評価テーブル。
- `js/storage.js`: localStorage による設定、保存済み対局、戦績の管理。
- `js/audio.js`: Web Audio API による背景音楽と効果音の生成。
- `js/game.js`: オセロのルール、着手処理、手番進行、スコア計算、保存処理。
- `js/ai.js`: Minimax AI、アルファベータ枝刈り、盤面評価。
- `js/ui.js`: DOM 描画、画面遷移、設定連携、オーバーレイ、レスポンシブ盤面サイズ調整。
- `js/main.js`: アプリケーションの起動処理。

## 繁體中文

### CSS

- `css/base.css`：設計變數、重置樣式、全域版面基礎、按鈕與表單基本樣式。
- `css/screens.css`：主畫面、難度選擇畫面，以及可重用的畫面與面板版面。
- `css/game.css`：棋盤、格子、棋子、計分面板、狀態列與遊戲中操作元件。
- `css/overlays-settings.css`：暫停、結果、規則說明 overlay、設定 UI、戰績 UI 與 toast。
- `css/responsive-motion.css`：關鍵影格動畫，以及平板與手機版響應式規則。

### JavaScript

- `js/constants.js`：棋盤常數、方向陣列與 AI 位置權重表。
- `js/storage.js`：localStorage 設定、存檔對局與戰績管理。
- `js/audio.js`：使用 Web Audio API 產生背景音樂與音效。
- `js/game.js`：黑白棋規則、落子處理、回合流程、計分與存檔連動。
- `js/ai.js`：Minimax AI、Alpha-Beta 剪枝與盤面評估。
- `js/ui.js`：DOM 渲染、畫面切換、設定綁定、overlay 與響應式棋盤尺寸。
- `js/main.js`：應用程式啟動流程。
