# 2048 完整版 Web 遊戲規格書

> 版本：v1.0.0 ｜ 日期：2026年4月27日 ｜ 平台：Web (HTML5 / CSS3 / JS)

---

## 1. 專案概述

### 1.1 專案簡介

本規格書定義了一款完整版 2048 Web 遊戲的所有功能、介面、技術實作細節。目標是打造一款具備主畫面、設定系統、完整音效、多種遊戲模式，並且支援桌面與行動裝置的高品質瀏覽器遊戲。

### 1.2 技術棧

| 項目 | 技術 |
|------|------|
| 前端框架 | Vanilla JavaScript (ES2022+) 或 React 18 |
| 樣式 | CSS3 + CSS Variables + Flexbox/Grid |
| 音效引擎 | Web Audio API（Tone.js 輔助） |
| 儲存 | localStorage + IndexedDB |
| 動畫 | CSS Transitions + requestAnimationFrame |
| 打包工具 | Vite 5.x |
| 測試 | Vitest + Playwright (E2E) |

---

## 2. 遊戲架構

### 2.1 頁面結構

採用 SPA（Single Page Application）架構，透過 JavaScript 控制畫面切換，共有以下主要頁面：

| 畫面 | 說明 |
|------|------|
| 主畫面 | 遊戲入口、快速開始、最高分展示 |
| 遊戲畫面 | 核心遊戲區域、即時分數、功能按鈕 |
| 設定畫面 | 音效、視覺主題、遊戲規則客製化 |
| 排行榜畫面 | 本地最高分紀錄、遊玩統計 |
| 說明畫面 | 遊戲規則說明、操作教學 |
| 暫停選單 | Overlay 覆蓋層，遊戲中可隨時呼出 |

## 3. 主畫面規格

### 3.1 視覺佈局

主畫面採用垂直置中佈局，元素由上至下依序為：

- 遊戲 Logo「2048」— 大字體、帶有陰影光暈效果
- 副標題「Slide & Merge」— 細體英文
- 最高分卡片區塊（顯示個人紀錄）
- 主要按鈕群：開始遊戲、繼續遊戲（若有存檔）、設定、排行榜、說明
- 頁尾版本號與音效快捷按鈕

### 3.2 按鈕功能定義

| 按鈕 | 功能 |
|------|------|
| 開始遊戲 | 進入遊戲畫面，使用目前設定初始化新局 |
| 繼續遊戲 | 若 localStorage 有存檔則恢復上一局（無存檔時灰化不可按） |
| 設定 | 跳轉至設定畫面 |
| 排行榜 | 跳轉至分數紀錄畫面 |
| 說明 | 跳轉至操作教學畫面 |

## 4. 核心遊戲規格

### 4.1 遊戲板

| 項目 | 規格 |
|------|------|
| 預設尺寸 | 4×4 格 |
| 可選尺寸 | 3×3、4×4、5×5、6×6（設定中選擇） |
| 初始方塊 | 遊戲開始時隨機生成 2 個方塊（值為 2 或 4） |
| 新方塊生成 | 每次移動成功後，隨機空格生成 1 個（90% 機率為 2，10% 為 4） |
| 勝利條件 | 預設出現 2048 數值的方塊（可在設定中調整） |
| 失敗條件 | 無任何合法移動方向時遊戲結束 |

### 4.2 移動與合併規則

- 玩家可向上、下、左、右四個方向滑動
- 所有方塊朝指定方向移動，直到碰到邊界或另一方塊
- 相同數值的相鄰方塊合併為其總和（例如：4 + 4 = 8）
- 每次移動中，每個方塊最多只能合併一次
- 合併後產生的分數立即加入目前分數
- 若移動後沒有任何方塊位移或合併，視為無效移動，不觸發生成新方塊

### 4.3 操作方式

| 裝置 | 操作 |
|------|------|
| 桌面鍵盤 | 方向鍵 ↑↓←→ 或 WASD |
| 桌面滑鼠 | 點擊遊戲區下方的方向按鈕 |
| 觸控裝置 | 滑動手勢（最小位移 30px 視為有效） |
| 快捷鍵 | R = 重新開始、Z = 復原、P = 暫停、M = 靜音 |

### 4.4 遊戲內 HUD 元件

| 元件 | 說明 |
|------|------|
| 目前分數 | 即時更新，合併時有 +N 浮現動畫後消失 |
| 最高分 | 從 localStorage 讀取，超越時即時更新並播放特殊音效 |
| 步數計數器 | 紀錄本局移動次數 |
| 計時器 | 可選功能，顯示本局已用時間（設定中開關） |
| 復原按鈕 | 最多復原 1 步（可在設定中擴充至 3 步） |
| 暫停按鈕 | 呼出暫停選單 Overlay |

## 5. 音效系統規格

### 5.1 音效清單

所有音效使用 Web Audio API 程序化生成或載入 `.ogg` / `.mp3` 資源，並提供 Fallback。

| 事件 | 音效描述 | 類型 | 備註 |
|------|----------|------|------|
| 方塊滑動 | 輕柔滑動聲 | SFX | 有方塊實際移動才觸發 |
| 方塊合併 | 清脆合成音，音高隨數值升高 | SFX | 每次合併獨立音高 |
| 新方塊生成 | 輕彈音效 | SFX | 每回合最多一次 |
| 破紀錄 | 短促勝利號角 | SFX | 首次超越最高分時觸發 |
| 達成 2048 | 歡呼 + 特效音樂 | Jingle | 可循環播放 |
| 遊戲結束 | 低沉下滑音 | SFX | — |
| 按鈕點擊 | UI 回饋音 | UI SFX | 所有按鈕共用 |
| 復原操作 | 回溯音效 | SFX | — |
| 背景音樂 | 輕柔電子氛圍樂 | BGM | 設定中可關閉 / 換曲 |

### 5.2 音量控制

- 主音量滑桿（0–100%）
- 音效（SFX）獨立音量控制
- 背景音樂（BGM）獨立音量控制
- 一鍵靜音按鈕（保留設定值，僅暫時靜音）
- 頁面隱藏時（切換分頁）自動暫停 BGM，返回時恢復

## 6. 設定系統規格

### 6.1 遊戲設定

| 設定項 | 說明 |
|--------|------|
| 棋盤大小 | 下拉：3×3 / 4×4 / 5×5 / 6×6，預設 4×4 |
| 勝利目標 | 下拉：512 / 1024 / 2048 / 4096 / 8192，預設 2048 |
| 復原次數 | 下拉：0次 / 1次 / 3次 / 無限，預設 1次 |
| 繼續遊戲 | Toggle：達到勝利目標後是否繼續而非強制結束，預設開啟 |
| 計時器 | Toggle：顯示本局計時，預設關閉 |
| 動畫速度 | 滑桿：慢 / 標準 / 快 / 無動畫，預設標準 |

### 6.2 視覺設定

| 設定項 | 說明 |
|--------|------|
| 主題配色 | 經典（米黃）/ 深色模式 / 海洋藍 / 森林綠 / 自訂 |
| 方塊字體 | 系統字體 / 圓角字體 / 像素字體 |
| 方塊圓角 | 滑桿 0–20px，預設 6px |
| 顯示網格線 | Toggle，預設關閉 |
| 高對比模式 | Toggle，用於視力輔助，預設關閉 |

### 6.3 音效設定

| 設定項 | 說明 |
|--------|------|
| 主音量 | 滑桿 0–100%，預設 80% |
| 音效音量 | 滑桿 0–100%，預設 70% |
| BGM 音量 | 滑桿 0–100%，預設 40% |
| BGM 曲目 | 下拉：環境音 / 輕電子 / 鋼琴 / 靜音 |
| 合併音效風格 | 音樂音階 / 電子合成 / 自然音，預設音樂音階 |

### 6.4 設定儲存規則

- 所有設定即時儲存至 localStorage
- 提供「恢復預設」按鈕，確認對話框後重置
- 設定頁面離開時自動儲存，無需手動點擊儲存

## 8. 排行榜與數據系統

### 8.1 本地紀錄

- 儲存最近 20 局的詳細紀錄
- 每筆紀錄包含：最終分數、最大方塊值、步數、用時、棋盤大小、日期
- 按分數排序的前 10 名最高分榜
- 統計數據：總局數、平均分、最長連勝、達成 2048 次數

### 8.2 本局統計

- 即時顯示：目前分數、移動次數、最大合併值
- 遊戲結束畫面顯示：本局排名、與個人紀錄的差距、進步提示

## 10. 技術實作規格

### 10.1 專案目錄結構

```
src/
├── core/
│   ├── game.js          # 遊戲邏輯核心（純函式，無 DOM 依賴）
│   └── board.js         # 棋盤狀態管理
├── ui/
│   ├── renderer.js      # DOM 渲染模組
│   └── animation.js     # 動畫控制器
├── audio/
│   ├── engine.js        # 音效引擎（Web Audio API）
│   └── tracks.js        # BGM 軌道管理
├── screens/
│   ├── home.js          # 主畫面邏輯
│   ├── game.js          # 遊戲畫面邏輯
│   └── settings.js      # 設定畫面邏輯
├── store/
│   ├── settings.js      # 設定 localStorage 讀寫
│   └── scores.js        # 分數 / 紀錄儲存
└── utils/
    └── input.js         # 鍵盤 / 觸控輸入處理

assets/
├── audio/
│   ├── sfx/             # 音效素材 .ogg / .mp3
│   └── bgm/             # 背景音樂素材
└── fonts/               # 自訂字體
```

### 10.2 效能目標

| 指標 | 目標值 |
|------|--------|
| 首屏載入時間（LCP） | < 2 秒 |
| 遊戲邏輯延遲 | < 16ms（60fps 動畫） |
| 音效觸發延遲 | < 50ms（Web Audio API） |
| Bundle 大小 | < 150KB gzip（不含音效素材） |
| 記憶體佔用 | < 50MB（含音效預載） |
