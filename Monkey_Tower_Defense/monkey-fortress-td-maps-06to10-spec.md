### 2.2 地圖說明

深海廢墟主題，引入全新的**雙端對沖機制**：敵人同時從地圖**左側與右側**兩個入口進入，相向而行，最終匯聚於地圖**中央**的出口。

### 3.2 地圖說明

### 4.4 動態路徑切換機制

```javascript
class DynamicPathManager {
  constructor() {
    this.currentAuxPath = 'A';   // 初始輔路
    this.switchInterval = 3;     // 每 3 波切換
  }

  onWaveStart(waveNumber) {
    if (waveNumber % this.switchInterval === 0) {
      this.switchAuxPath();
    }
  }

  switchAuxPath() {
    const prev = this.currentAuxPath;
    this.currentAuxPath = prev === 'A' ? 'B' : 'A';

    // 動畫：傳送帶箭頭旋轉 + 路徑格顏色漸變（500ms 過渡）
    this.playTransitionAnimation(prev, this.currentAuxPath);

    // 通知玩家
    EventBus.emit('PATH_SWITCHED', {
      from: prev,
      to:   this.currentAuxPath,
      wave: waveNumber
    });
  }
}
```

**切換時的 UI 提示：**
```
⚙️  傳送帶重新配置！輔路切換至 B 段！
```
顯示於畫面頂部，持續 2 秒，橘黃色文字。

### 5.4 迷霧系統機制

```javascript
class FogSystem {
  constructor(mapGrid) {
    // 初始化：所有非路徑起始段皆為霧
    this.fogGrid = mapGrid.map(row =>
      row.map(cell => cell.type === 'F' ? 'hidden' : 'visible')
    );
    this.visionRadius = 2;   // 塔的視野半徑（格）
  }

  // 放置塔時揭露周圍格子
  revealAroundTower(towerX, towerY) {
    for (let dy = -this.visionRadius; dy <= this.visionRadius; dy++) {
      for (let dx = -this.visionRadius; dx <= this.visionRadius; dx++) {
        const dist = Math.abs(dx) + Math.abs(dy);
        if (dist <= this.visionRadius) {
          this.fogGrid[towerY + dy][towerX + dx] = 'visible';
        }
      }
    }
  }

  // 探照燈塔視野半徑更大（4 格）
  revealAroundSpotlight(x, y) {
    this.revealCircle(x, y, 4);
  }

  // 敵人在霧中時的渲染：塔仍攻擊，但玩家看到的是問號圖示
  renderEnemy(enemy, ctx) {
    if (this.fogGrid[enemy.gridY][enemy.gridX] === 'hidden') {
      ctx.fillText('❓', enemy.renderX, enemy.renderY);
    } else {
      enemy.draw(ctx);
    }
  }
}
```
