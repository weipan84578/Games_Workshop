## 2. 地圖資料結構

### 2.1 地圖 JSON 頂層結構

```javascript
MapData = {
  id: String,                   // 'map01'
  name: String,                 // '叢林小徑'
  nameEn: String,               // 'Jungle Path'
  theme: String,                // 'jungle' | 'desert' | 'snow' | 'lava' | 'sky'
  difficulty: Number,           // 1~5
  gridWidth: Number,            // 格子欄數
  gridHeight: Number,           // 格子列數
  startPoint: { x, y },        // 敵人入口格（地圖邊緣）
  endPoint: { x, y },          // 敵人出口格（地圖邊緣）
  paths: Path[],                // 路徑定義（多路徑地圖有多個）
  grid: String[],               // 逐列字元地圖（見格子代碼）
  waves: Wave[],                // 波次定義
  startGold: Number,            // 初始金幣
  startLives: Number,           // 初始生命
  background: ThemeColors,      // 主題配色
  music: String,                // 使用的背景音樂主題
  unlockCondition: Object       // 解鎖條件
}
```

### 2.2 格子代碼（grid 字元）

```
'.' = 空地（可放塔）
'#' = 路徑（不可放塔）
'>' = 路徑（視覺方向提示：向右）
'<' = 路徑（向左）
'^' = 路徑（向上）
'v' = 路徑（向下）
'S' = 敵人起點
'E' = 敵人終點
'X' = 障礙物（不可放塔，非路徑）
'W' = 水域（不可放塔，非路徑）
'R' = 岩石（不可放塔，非路徑）
'T' = 裝飾樹木（不可放塔）
'B' = 建築裝飾（不可放塔）
'2' = 路徑分叉點（雙路徑交匯）
```

### 2.3 Path 結構

```javascript
Path = {
  id: String,          // 'main' | 'upper' | 'lower' | 'left' | 'right'
  waypoints: [         // 路徑轉折點（格子座標）
    { x: 0, y: 5 },   // 起點
    { x: 4, y: 5 },
    { x: 4, y: 2 },
    ...
    { x: 15, y: 8 }   // 終點
  ]
}
```

### 2.4 主題配色結構

```javascript
ThemeColors = {
  bgColor: String,        // 地圖背景底色
  pathColor: String,      // 路徑顏色
  pathEdgeColor: String,  // 路徑邊緣線
  emptyColor: String,     // 空地顏色
  obstacleColor: String,  // 障礙物顏色
  accentColor: String,    // 裝飾強調色
  uiTint: String          // HUD 主題色調
}
```

## 4. Map 02 — 沙漠蜿蜒 (Desert Serpent)

### 4.1 基本資訊

| 項目 | 值 |
|------|----|
| ID | `map02` |
| 難度 | ⭐⭐ 普通 |
| 網格 | 18 × 13 |
| 路徑數 | **2（上下雙路徑，分叉合流）** |
| 初始金幣 | 130 |
| 初始生命 | 18 |
| 波次數 | 12 |
| 音樂主題 | `desert_theme` |

### 4.2 地圖說明

沙漠地形引入**雙路徑分叉機制**。敵人從左側同一入口出發，在地圖中央**分成上下兩條路**，各自蜿蜒後在右側重新匯合，共同走向終點。玩家必須同時顧及兩條路，資源分配是關鍵。

```

  2   .  .  .  >  >  >  >  >  >  >  >  >  v  .  .  .  .  .  ← 上路
  3   S  >  2  .  .  .  .  .  .  .  .  .  .  .  .  .  2  E
  4   .  .  .  >  >  >  >  >  >  >  >  >  ^  .  .  .  .  .  ← 下路
  5   .  R  .  .  .  .  .  R  .  .  .  .  .  .  R  .  .  .
```

**精確 waypoints（上路）：**
```javascript
pathUpper: {
  waypoints: [
    { x: 0,  y: 3 },   // 共同入口
    { x: 2,  y: 3 },   // 分叉點
    { x: 2,  y: 2 },   // 進入上路
    { x: 12, y: 2 },   // 上路向右
    { x: 12, y: 3 },   // 回到中線
    { x: 16, y: 3 },   // 合流點
    { x: 17, y: 3 }    // 終點 E
  ]
}

pathLower: {
  waypoints: [
    { x: 0,  y: 3 },
    { x: 2,  y: 3 },   // 分叉點
    { x: 2,  y: 4 },   // 進入下路
    { x: 12, y: 4 },   // 下路向右
    { x: 12, y: 3 },   // 回到中線
    { x: 16, y: 3 },
    { x: 17, y: 3 }
  ]
}
```

### 4.4 雙路徑敵人分配邏輯

```javascript
// 波次中奇數敵人走上路，偶數走下路（或依特殊波次設計全走單路）
enemy.assignPath = (index, wave) => {
  if (wave.forcePath) return wave.forcePath;
  return index % 2 === 0 ? 'upper' : 'lower';
}
```

```javascript
// 根據路徑方向繪製裝飾箭頭
function drawPathArrow(ctx, x, y, direction, tileSize) {
  const cx = x * tileSize + tileSize / 2;
  const cy = y * tileSize + tileSize / 2;
```
