## 二、 敵人資訊 (Enemies)

氣球分為基礎型、護甲型與特殊機制型，針對不同塔有免疫或弱點關係。

### 1. 基礎與護甲氣球
*   **🔵 藍色**：基礎敵人 (1HP)。
*   **🟢 綠色**：速度較快，擊破後分裂出藍色。
*   **🟡 黃色**：極速型。
*   **🔴 紅色**：血量較厚 (3HP)。
*   **⚫ 黑色**：**免疫爆炸**傷害。
*   **🔩 鉛色**：重護甲，**免疫冰凍與一般飛鏢**。
*   **🔩 鋼殼**：重甲且**抗爆炸**，但**弱點為音波** (鈴鐺猴)。

### 2. 特殊機制氣球
*   **⚡ 電磁**：抗穿甲，**弱點為電擊**。
*   **🧱 絕緣**：**免疫電擊與冰凍**，弱點為火焰。
*   **🍃 淨化**：**免疫毒素**，帶有淨化光環解除附近敵人的毒素。
*   **💨 疾風**：極速且**抗範圍攻擊**，適合用飛鏢猴攔截。
*   **🐢 龜甲**：超重甲且抗減速，外殼破碎後會進入暴速模式。
*   **🌀 時空**：抗冰凍，每 5 秒會向前方瞬間跳躍 3 格。
*   **🔄 再生**：每 3 秒自動回復生命值。
*   **💀 詛咒**：擊殺它的塔會被減速 5 秒。
*   **🪞 鏡像**：進入射程後分裂成 3 個分身。
*   **💣 自爆**：殘血時倒數爆炸，扣除玩家生命並傷及周圍敵人。
*   **🌑 暗影**：迷霧中無敵，需視野偵測才能攻擊。
*   **💀 BOSS**：極高血量，具備多重抗性，通過終點扣除大量生命。

## 四、 玩法與操作 (Gameplay)

### 1. 核心規則
*   **生存目標**：阻止氣球到達終點。初始生命依難度而定 (通常 10-20)，歸零則失敗。
*   **資源管理**：透過擊破氣球、完成波次、提前發動波次或賣出塔獲得金幣。
*   **波次預覽**：發動前可看見下一波敵人的免疫特性（如 ❌穿甲、❌爆炸）。

### 2. 控制操作
*   **放置/升級**：點擊底部塔列選購，點擊地圖空地放置。點擊場上的塔可開啟升級面板。
*   **遊戲速度**：提供 **1× / 2× / 3× / 4× / 5×** 五段切換（地獄難度禁用）。
*   **自動波次**：可開啟「⚡ 自動：ON」，在前一波清空後自動發動下一波。
*   **提前獎勵**：倒數結束前手動發動波次，可獲得 **+10 金幣**。

### 3. 模式與難度
*   **難度模式**：簡單 (0.7x 分數)、普通 (1x)、困難 (1.5x)、地獄 (2.5x，禁用賣塔與加速)。
*   **關卡長度**：速戰速決 (0.6x 波次)、標準、史詩長征 (1.8x 波次)。

### 4. 密技系統 (Cheat Code)
*   在遊戲進行中輸入：**← ← → → ←** (左左右右左)。
*   效果：立即獲得 **+9999 金幣**。
*   備註：使用密技後該局不計入排行榜。
