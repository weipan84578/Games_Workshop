# Checkers Master

純前端 Web 版跳棋遊戲，依 `checkers_spec.md` 實作。無需後端與外部套件，可直接開啟 `index.html` 遊玩。

## 執行方式

在瀏覽器開啟 `index.html`。

## 已包含

- 8x8 跳棋棋盤、強制吃子、連續跳躍、升王、勝負和平局判定
- 單人對 AI，支援簡單、普通、困難三種 Minimax 深度
- 設定畫面、暫停選單、遊戲結束統計與歷史戰績
- 依規格書第 9 章整理的遊戲說明 Modal
- localStorage 保存設定與戰績
- Web Audio API 合成背景音樂與互動音效
- 桌機與手機響應式版面、鍵盤基本操作、ARIA 標記
